\# UI STANDARD - STARLEX



Todas las pantallas deben seguir esta estructura obligatoria:



\## Layout Base



1\. HEADER FIJO

\- Logo (pequeño, lado izquierdo)

\- Nombre app: "STARLEX"

\- Avatar de usuario (lado derecho), este desplegara un menu con mas opciones adicionales a los botton navigation



2\. TÍTULO DE PANTALLA

\- Texto en mayúsculas

\- Ejemplo: "PROCESADOS", "CALENDARIO"

\- Debe ser claro y visible

\- Debe estar en el Header



3\. CONTENIDO

\- Sección dinámica

\- Evitar Scroll

\- Componentes reutilizables



4\. BOTTOM NAVIGATION

\- Iconos:

&#x20; - Inicio

&#x20; - Calendario

&#x20; - Clientes (Procesados)

&#x20; - Finanzas

&#x20; - Ajustes



\## Estilo visual



\- Diseño limpio y profesional

\- Colores por estado:

&#x20; - Amarillo → Indagación

&#x20; - Azul → Imputados

&#x20; - Rojo → Juicio



\- Tarjetas:

&#x20; - Bordes redondeados

&#x20; - Sombra ligera

&#x20; - Información clara y compacta



\## Reglas obligatorias



\- No crear headers diferentes

\- No crear footers diferentes

\- No duplicar navegación

\- Toda pantalla usa el mismo layout base

\- Toda pantalla usa el mismo logo
