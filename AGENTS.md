# LOGS POR PANTALLA

Por cada pantalla que crees o modifiques, crea o actualiza un archivo en `log/` con el mismo nombre de la ruta (ej: `log/index.txt`, `log/LoginScreen.txt`). Usa formato `.txt` liviano.

Estructura del log:
```
<nombre_archivo> — <título_pantalla>
========================================
- <acción 1>
- <acción 2>
```

Propósito: mantener memoria ligera de todo lo implementado por pantalla.
