- UI → + ux_ui + theme

---

# 📱 CREAR PANTALLA

Usa context + architecture + ux_ui + mobile_react_native + theme

Crear pantalla de [nombre de la pantalla]

Requisitos:
- UI limpia
- Usar services
- NO lógica en components

OBLIGATORIO:
- Usar Layout global (Header + BottomTabs)
- No crear headers propios
- No duplicar navegación
- Solo implementar contenido interno

ESTILO:
- Usar colors, spacing y typography (Design System)
- No hardcodear estilos
- Seguir estándar definido en ux_ui.md
